mas como a ciência de dados pode ajudar a resolver o problema?

Problema de Negócio

“Uma das grandes empresas do setor varejista gasta milhões e milhões anualmente com marketing em diversos meios de comunicação e sempre a busca pela relação inversa de gasto com retorno. Sabendo disso, uma equipe de cientistas de dados dessa empresa recebeu uma solicitação de ajuda do RH para resolver um conflito entre o setor financeiro com o de marketing, visto que o setor financeiro sempre quer repassar um orçamento menor para o marketing e o marketing pede um orçamento sempre maior. Sendo assim o RH solicitou que os cientistas de dados encontrem uma solução usando dados para deixar essa negociação em um campo com base histórica.”

Como resolver esse problema com a ciência de dados

A melhor maneira de ambos chegarem a um acordo é o setor financeiro junto ao de marketing saberem como o gasto com anúncio em diversos meios de comunicação pode gerar de retorno para ambos trabalharem juntos no acordo orçamentário. Para isso um modelo de regressão deve ser utilizado por retornar um valor numérico, que será a estimativa de retorno com o dinheiro investido em marketing

Descrição do trabalho realizado

A primeira etapa é avaliar o histórico de qual foi o orçamento, como ele foi distribuído entre os meios de comunicação (rádio, televisão e jornal) e como podemos criar mais informações para o modelo de machine learning aprender melhor com os dados. Com isso podemos então avaliar qual a relação que cada um teve com os ganhos.

Qual modelo utilizar? Como aparentemente a base de dados é simples, dois modelos serão testados, um modelo mais complexo de rede neural e outro mais simples de regressão polinomial. A avaliação dos resultados será feita utilizando as medidas de erro RMSE e R2, pois acredito que ambos são as duas métricas para regressão com os resultados mais compreensíveis.
