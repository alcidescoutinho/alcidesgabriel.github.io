Problema de Negócio

“Mary é uma cientista de dados recém-formada que reside no nordeste brasileiro e deseja ingressar no mercado de trabalho, mas Mary não tem noção de como anda o mercado brasileiro e o que as vagas de emprego estão exigindo dos profissionais. Então Mary começa a vasculhar sites de oferta de emprego para começar a entender, mas vê que são muitas informações e que essa análise pode levar muito tempo, então ela começa a pensa se há uma maneira de aplicar as suas habilidades de cientista de dados para resolver o problema.”

Como resolver esse problema com a ciência de dados

Para coletar e analisar esses dados, Mary terá que fazer 3 etapas diferentes. A primeira é a web scraping, que em tradução livre significa “Raspagem de Dados”, que é a ação de coletar os dados da web de forma automática. Como esses dados são, sem sua maioria, informações textuais, será necessário utilizar então o NLP (natural language processing) que é a forma de trabalhar com esses dados.

Descrição do trabalho realizado

O modelo foi desenvolvido em 3 etapas para resolver esse problema de Mary, sendo em ordem: Coletando os dados, Tratamento dos dados e Análise em POWER BI.

Para coletar os dados foi utilizado a biblioteca selenium, que “cria um robô”, no qual ele interage com o site com as ordens dadas, sendo assim é possível configurar as ações de coleta de dados que o robô irá executar e retornar o que foi desejado.

A coleta desses dados apresenta muitos dados compilados e sujos, então é necessário todo o tratamento de feature engineering para poder transformar essas informações em dados úteis para análise.

O uso de uma ferramenta de BI, como no caso o power BI, é para ter uma análise mais dinâmica e exploratória mais eficiente, afinal de conta é possível visualizar diversas informações e aplicar diversos filtros com o BI para compreensão melhor dos dados de vagas.

Abaixo você pode encontrar os botões que darão acesso tanto aos códigos de coleta e tratamento, assim como para o relatório de BI, mas caso não tenha conta de acesso também disponibilizei em pdf para a visualização do trabalho realizado. Os códigos apresentam cada etapa bem explicada como também demonstra o resultado final alcançado.
